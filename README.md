# pakasa-io/di

A pragmatic dependency injection library for Go, focused on:

- excellent developer experience
- clear and actionable errors
- predictable lifetimes and scoping
- minimal magic, explicit behavior

This library is intended for production services, CLIs, workers, and long-running
processes where correctness and maintainability matter more than cleverness.

go get github.com/pakasa-io/di@latest
